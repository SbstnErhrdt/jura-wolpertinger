Installation

Jura Wolpertinger installieren

Lade die passende Datei direkt herunter. Die App läuft lokal auf deinem Rechner und speichert Klausuren offline-ready.

Download auswählen

Windows

  1. Windows-Installer herunterladen.
  2. Installer starten.
  3. Dem Installationsdialog folgen.
  4. Die App über Startmenü oder Desktop-Verknüpfung öffnen.

Windows SmartScreen kann bei neuen oder unsignierten Apps warnen. Für öffentliche Releases sollte der Installer mit einem vertrauenswürdigen Code-Signing-Zertifikat oder Microsoft Trusted Signing signiert werden.

macOS

  1. Passende DMG-Datei herunterladen: Apple Silicon oder Intel.
  2. DMG öffnen.
  3. `Jura Wolpertinger.app` in den Programme-Ordner ziehen.
  4. Die App aus `Programme` starten.

Für öffentliche macOS-Releases sind Developer-ID-Signierung, Hardened Runtime und Notarisierung praktisch notwendig, damit Gatekeeper die App normal startet.

Linux

  1. Linux-AppImage herunterladen.
  2. Ausführbar machen: `chmod +x "Jura Wolpertinger-x64-linux.AppImage"`.
  3. Starten: `./Jura\ Wolpertinger-x64-linux.AppImage`.

Für AppImages gibt es kein zentrales Vendor-Zertifikat wie bei macOS oder Windows. Empfehlenswert sind SHA256-Checksums und optional GPG-Signaturen.

Brauchen wir Zertifikate?

Für interne Tests nicht zwingend. Für öffentliche Releases ja: macOS sollte signiert und notarisiert werden, Windows sollte code-signiert werden, Linux sollte mindestens Checksums bekommen.

Die ausführliche Maintainer-Anleitung steht zusätzlich als Markdown-Datei im Repository: installation.md.
