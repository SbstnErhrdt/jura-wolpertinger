Installation

Desktop-App installieren

Wähle die passende Datei für dein System und installiere Jura Wolpertinger wie eine normale Desktop-App.

macOS

  1. Passende DMG-Datei laden: Apple Silicon für M1/M2/M3/M4, Intel für ältere Macs.
  2. DMG öffnen und Jura Wolpertinger.app in den Programme-Ordner ziehen.
  3. App aus dem Programme-Ordner starten.

Windows

  1. Windows-Installer herunterladen.
  2. Installer starten und den Schritten folgen.
  3. Jura Wolpertinger über das Startmenü öffnen.

Linux

  1. AppImage herunterladen.
  2. Datei ausführbar machen.
  3. AppImage starten.

Online-Version

Die Web-App läuft unter app.jura-wolpi.de. Dort ist ein kostenloser Account erforderlich. Die Desktop-App kann lokal genutzt und später mit der Online-Version verbunden werden.
